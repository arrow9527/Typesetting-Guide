間距

類別使用下列格式 {屬性}{方向}-{尺寸}。

屬性:

  • m 對於 margin
  • p 對於 padding

方向:

  • t 對於上(margin-top 或 padding-top)
  • b 對於下(margin-bottom 或 padding-bottom)
  • l 對於左(margin-left 或 padding-left)
  • r 對於 margin-right 或 padding-right
  • x 對於 x軸 (*-left 或 *-right)
  • y 對於 y軸 (*-top 或 *-bottom)
  • 空白 對於四方(margin 或 padding)

頁面大標題

為了切齊區域內元素及避免margin重疊，上距設定在父層.page-heading下距則在子元素<hr>。
整個區域向上隔12px，標題文字向下離分隔線12px(分隔下向上隔12px)，整個區域向下離16px(分隔線向下離16px)。
