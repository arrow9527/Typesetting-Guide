加入disabled屬性使其禁用，加入.is-invalid類別呈現紅色警告框


<div class="form-group row">
  <!-- Disabled -->
  <label class="col-1 col-form-label">標題</label>
  <div class="col-4">
    <input class="form-control" placeholder="Disabled input" disabled>
  </div>
  <div class="col-1"></div>
  <!-- invalid -->
  <label class="col-1 col-form-label">標題</label>
  <div class="col-4">
    <input class="form-control is-invalid" placeholder="invalid">
  </div>
</div>

Dropdown

使用<select>標籤styleselect.js將會產生選單樣式化的DOM，元素外框容器則依select給的類別定義，使用.custom-select。預設尺寸高度為38px，較小尺寸的高度為32px，加入.custom-select-sm類別使用。


<select class="custom-select mb-3">
  <option selected>全部</option>
  <option value="1">成功</option>
  <option value="2">失敗</option>
  <option value="3">執行中</option>
</select>
<!-- 小尺寸 -->
<select class="custom-select custom-select-sm">
  <option selected>全部</option>
  <option value="1">成功</option>
  <option value="2">失敗</option>
  <option value="3">執行中</option>
</select>

上面範例將產生一DIV容器包裹隱藏的原select跟樣式化的元素


<!-- 下拉容器 -->
<div class="custom-select mb-3">
  <!-- 隱藏的select -->
  <select class="custom-select mb-3 s-hidden">
        <option selected="">Open this select menu</option>
        <option value="1">One</option>
        <option value="2">Two</option>
        <option value="3">Three</option>
    </select>
    <!-- 顯示目前選項 -->
    <div class="">Open this select menu</div>
    <!-- 所有選項 -->
    <ul class="options" style="display: none;">
        <li rel="Open this select menu">Open this select menu</li>
        <li rel="1">One</li>
        <li rel="2">Two</li>
        <li rel="3">Three</li>
    </ul>
</div>

Radio button

將.image-check做容器使用，通過radio_checkbox.js，容器內的Radios將自動產生樣式化元素取代瀏覽器原生。


<div class="image-check">
  <input type="radio" id="exampleRadios0" name="exampleRadios">
  <label for="exampleRadios0">Default radio</label>
  <input type="radio" id="exampleRadios1" name="exampleRadios" checked>
  <label for="exampleRadios1">Default radio</label>
  <input type="radio" id="exampleRadios2" disabled>
  <label for="exampleRadios2">Disabled radio</label>
  <input checked="checked" type="radio" id="exampleRadios3" disabled>
  <label for="exampleRadios3">Disabled radio</label>
</div>
